## 🚀 Vercel 배포 방법

### 단계 1: Vercel 로그인
1. https://vercel.com 접속
2. GitHub 계정으로 로그인

### 단계 2: 프로젝트 Import
1. "Add New Project" 버튼 클릭
2. "Import Git Repository" 선택
3. 리포지토리 검색: `paulslife2017-hue/midas`
4. "Import" 버튼 클릭

### 단계 3: 프로젝트 설정
```
Project Name: yonsei-midas-dental (또는 원하는 이름)
Framework Preset: Other (정적 HTML)
Root Directory: ./
Build Command: (비워두기 - 정적 사이트라 필요 없음)
Output Directory: (비워두기)
Install Command: (비워두기)
```

### 단계 4: 환경 변수 (선택사항)
필요 없음 - 정적 사이트입니다

### 단계 5: Deploy!
"Deploy" 버튼 클릭하면 자동으로 배포됩니다 (약 1-2분 소요)
